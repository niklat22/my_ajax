# Rails bootstrap with ajax

This app use for rails bootstrep with ajax use.


# Prerequisites
  * Rails 5.2+
  * Ruby 2.3.1+
  * PostgreSQL 10.10
  * bootstrap 3.4.1

# Installing & Serving
  1. install dependencies

    `bundle install`

  2. Create db and migrate

    `rake db:create`
    `rake db:migrate`

Then run server.
